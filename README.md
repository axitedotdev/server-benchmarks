# Server Benchmarks
### axitetronic.sh
A fork of https://github.com/joshtronic/server-benchmarks that performs a CPU benchmark, memory benchmark, disk benchmark within SQL, and a multi-threaded network benchmark with the official Speedtest CLI software. Outputs all of the results in text format within a ZIP file.

### memory-benchmark.sh
A script that throughly benchmarks the memory performance of a system utilizing sysbench and outputs all of the results in text format within a ZIP file.
