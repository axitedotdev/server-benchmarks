# Server Benchmarks
### axitetronic.sh
A fork of https://github.com/joshtronic/server-benchmarks that performs a CPU benchmark, memory benchmark, disk benchmark within SQL, and a multi-threadewd network benchmark with the official Speedtest CLI software.

### memory-benchmark.sh
A script created to throughly benchmark the memory performance of a system utilizing sysbench and output all of the results in text format within a ZIP file.
